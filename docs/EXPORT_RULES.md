# 导入导出规则

## 总原则

导入导出是业务证据链的一部分，不能只做页面按钮。

## Excel 导入规则

1. 导入前必须保存原始文件。
2. 每一行原始 Excel 数据应保留原始 JSON 快照。
3. 导入后必须进入预览页，不允许直接进入正式业务表。
4. 预览页应区分：校验通过、需要确认、校验失败。
5. 校验失败的数据不得提交。
6. 导入批次必须记录导入人、导入时间、文件名、模板类型、总行数、成功行数、失败行数。

## 常见校验错误

- 违标编码不存在
- 编码与性质不匹配
- 工号不存在
- 工号与姓名不一致
- 责任部门不存在
- 违章日期为空
- 拟考核内容为空
- 疑似重复导入

## Excel 导出规则

1. 所有列表页、预览页、结果库、日志页原则上都支持导出。
2. 导出权限独立于查看权限。
3. 所有导出必须由后端校验导出权限。
4. 所有导出必须写导出日志。
5. 导出文件应记录导出人、导出时间、导出条件、数据量。

## 每日 LKJ 音视频违标公示导出特殊规则

- 标题包含提报日期。
- 表格列只导出“违章发生日期”，不单独导出“提报日期”列。
- 不导出分数、扣款字段。
- 导出表格应接近 Excel 公示格式。

## 导出模板建议

后续应支持导出模板配置：

- 模板名称
- 业务类型
- 标题规则
- 副标题规则
- 列顺序
- 列显示名称
- 是否显示导出人
- 是否显示导出时间
