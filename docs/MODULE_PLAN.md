## 推荐开发顺序

1. 初始化项目骨架。
2. 建立系统权限底座。
3. 建立公共信箱、待办、附件、业务日志、导入导出能力。
4. 建立基础数据：人员、组织、编码字典、导入导出模板。
5. 实现每日 LKJ 音视频违标公示样板业务。
6. 做权限审查、日志审查、导出审查。
7. 再扩展其他业务模块。
